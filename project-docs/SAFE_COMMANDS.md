# Allowed Operations
- profile read
- profile save
- profile import/export
- settings read
- shot history read
- statistics read
- analyzer read
# Blocked Operations
- process:start
- process:stop
- grinder:start
- steam:start
- water:start
- ota:start
- autotune:start
- raw websocket passthrough
# Core Rule
GaggiGo is not remote machine control software.
